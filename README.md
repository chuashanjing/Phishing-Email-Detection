### Phishing Email Detection
#
A program that determines the legitimacy of emails using machine learning. The program will be trained with trusted and non-trusted email domains, suspicious and non-suspicious keywords, and URLS. It uses keyword position scoring and distance checks to calculate a risk score for the email.
